KEMA Sales Order and Item Fulfillment Upload (NetSuite)

This process explains how to prepare and upload the required files into the KEMA Sales Order and Item Fulfillment Processor in NetSuite.

 

Generate the Tracking File (ShipStation)

  1. In ShipStation, navigate to: Shipments > KEMA SO
  2. Apply the Order Date filter. Select all orders for the selected date range.
  3. Click Export Shipments.

Export Settings

-          Which records would you like to export? Export all n orders in the current view

-          How would you like to export these? Export shipment line items

-          Export view: Export my current view

  1. Download and save the exported file.

 

 

Generate the PO Number File (NetSuite)

  1. In NetSuite, go to: Vendors > Purchases > Enter Purchase Orders > List
  2. Set the following filters:
    • View: PO Team Purchase Order Order Details
    • Type: Purchase Order
  3. (Optional) Adjust the date range to generate a smaller file.
  4. Click the Excel icon to export the PO file.

 

 

Upload Files to the Processor

  1. Upload the following into the processor:
    • Fulfilled SKUs file
    • Tracking file
    • PO Number file
  2. Manually add the PO Number if required.
  3. Confirm all files are successfully uploaded.

 

Verify Processor Output

  • Once all files are uploaded, the processor should populate the KEMA Sales Order table.
  • Review the data to ensure:
    • Orders match across all files
    • Tracking numbers and PO numbers are properly linked

         Add a file name then click Download Report. Change file type to CSV.

 

 

KEMA Sales Order Upload (NetSuite)

 

  1. Once the Sales Order CSV file is ready, go to NetSuite.
  2. Navigate to: Setup > Import/Export > Saved CSV Imports

  1. Under Filters, change Record Type to Sales Order.
  2. Select KEMA SO Template.

 

Upload Steps

  1. Upload the prepared CSV file.
  2. Click Next.
  3. Mapping is already configuredclick Next, then Run to complete the upload.

 

 

KEMA Item Fulfillment (NetSuite)

 

Generate Sales Order Reference File

  1. In NetSuite, go to:
    Sales > Enter Sales Orders > List
  2. Set the following:
    • View: For SO Search with details (Wholesale)
    • Customer: Sub Of
  3. Click the Excel icon to export the file.

 

Note: This file is required to retrieve the Document Number, which is needed to generate the Sales Order Number for Item Fulfillment.

 

 

Item Fulfillment File Preparation

1.      Return to the KEMA Sales Order and Item Fulfillment Processor.

2.      Upload the exported Sales Order file.

The processor will automatically populate:

-          SO Number

-          Document Number

3.      Apply filters to hide unnecessary columns.

4.      Enter a file name, then click Download Export.

5.      Change the file type to CSV.

 

Item Fulfillment Upload (NetSuite)

1.      In NetSuite, go to: Setup > Import/Export > Saved CSV Imports

2.      Under Filters, change Record Type to Item Fulfillment.

3.      Select KEMA SO Item Fulfillment v2.

4.      Mapping is already configuredclick Next, then Run to complete the upload.
