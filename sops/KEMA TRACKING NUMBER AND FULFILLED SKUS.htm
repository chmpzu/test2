KEMA Tracking Numbers & Fulfilled SKUs

Part 1: Fulfilled SKUs File

 

1.      Export shipped KEMA orders from ShipStation

·         Navigate to Shipped > KEMA Int’l.

·         Apply the Order Date filter.

 

2.      Export the orders

·         Select all orders in the filtered view.

·         Click Other Actions > Export Orders > Export.

 

3.      Export settings

·         Which records to would you like to export? – Export all n records in the current view

·         How would you like to export these? – Export order line items

·         What format do you want exported? – KEMA ORDER / DS Format

 

4.      Validation

·         Cross-check the orders and items from the exported file against ShipStation (SS) and PMB to ensure all data matches.

 

Part 2: Tracking Numbers File

 

1.      Access Raw Data Exports

·         Go to Insight > Reports > Raw Data Exports.

·         Select Shipped Orders.

 

2.      Configure export

·         Edit the date range as needed.

·         Select the appropriate store.

·         Set Export To: Excel.

·         File will automatically download after clicking Excel

 

 

3.      Validation

·         Cross-check the order numbers from the exported file against ShipStation (SS) and PMB to confirm all records match.

 

Notes / Best Practices

·         Always verify both order-level and item-level data.

·         Ensure date ranges match across ShipStation, PMB, and exported files.

·         Address discrepancies before final submission.
